# nm - List symbols from object files



## Overview

The `nm` command list symbols from object files.

## Syntax

```bash
nm [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
nm

# Get help
nm --help

# Check version
nm --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `nm` command is part of the `development` category in Linux systems. It provides essential functionality for list symbols from object files.

Refer to the manual page (`man nm`) for detailed information about all available options and usage patterns.

# mysqldump - Database backup utility



## Overview

The `mysqldump` command database backup utility.

## Syntax

```bash
mysqldump [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
mysqldump

# Get help
mysqldump --help

# Check version
mysqldump --version
```

- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

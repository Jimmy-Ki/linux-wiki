# Text Processing Commands

This section contains commands for text manipulation, pattern searching, and text file processing.

## Commands Included:
- **awk** - Pattern scanning and processing language
- **col** - Filter reverse line feeds from input
- **colrm** - Remove columns from input
- **comm** - Compare two sorted files line by line
- **csplit** - Split a file into context-determined pieces
- **ed** - Line-oriented text editor
- **egrep** - Print lines matching a pattern
- **ex** - Line-oriented text editor
- **fgrep** - Print lines matching a pattern
- **fmt** - Simple optimal text formatter
- **fold** - Wrap each input line to fit in specified width
- **grep** - Print lines matching a pattern
- **ispell** - Interactive spelling checker
- **jed** - JED editor
- **joe** - Joe's Own Editor
- **join** - Join lines of two files on a common field
- **look** - Display lines beginning with a given string
- **mtype** - Display contents of MSDOS files
- **pico** - Simple text editor in the style of the Pine Composer
- **rgrep** - Recursive grep
- **sed** - Stream editor
- **sort** - Sort lines of text files
- **spell** - Report spelling errors
- **tr** - Translate or delete characters
- **uniq** - Report or omit repeated lines
- **vi** - Visual editor
- **wc** - Print byte, word, and line counts

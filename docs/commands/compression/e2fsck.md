# e2fsck - Check ext2/ext3/ext4 filesystems



## Overview

The `e2fsck` command check ext2/ext3/ext4 filesystems.

## Syntax

```bash
e2fsck [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
e2fsck

# Get help
e2fsck --help

# Check version
e2fsck --version
```

## Common Options

- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

# e2label - Change ext2/ext3/ext4 filesystem label



## Overview

The `e2label` command change ext2/ext3/ext4 filesystem label.

## Syntax

```bash
e2label [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
e2label

# Get help
e2label --help

# Check version
e2label --version
```

- `-V, --verbose` - Enable verbose output (if supported)

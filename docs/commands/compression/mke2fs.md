# mke2fs - Create ext2/ext3/ext4 filesystem



## Overview

The `mke2fs` command create ext2/ext3/ext4 filesystem.

## Syntax

```bash
mke2fs [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
mke2fs

# Get help
mke2fs --help

# Check version
mke2fs --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

The `mke2fs` command is part of the `compression` category in Linux systems. It provides essential functionality for create ext2/ext3/ext4 filesystem.

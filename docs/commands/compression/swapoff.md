# swapoff - Disable devices for paging and swapping



## Overview

The `swapoff` command disable devices for paging and swapping.

## Syntax

```bash
swapoff [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
swapoff

# Get help
swapoff --help

# Check version
swapoff --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `swapoff` command is part of the `compression` category in Linux systems. It provides essential functionality for disable devices for paging and swapping.

Refer to the manual page (`man swapoff`) for detailed information about all available options and usage patterns.

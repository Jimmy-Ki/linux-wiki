# source - Execute commands from file



## Overview

The `source` command execute commands from file.

## Syntax

```bash
source [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
source

# Get help
source --help

# Check version
source --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `source` command is part of the `system-info` category in Linux systems. It provides essential functionality for execute commands from file.

Refer to the manual page (`man source`) for detailed information about all available options and usage patterns.

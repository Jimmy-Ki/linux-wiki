# history - Display command history



## Overview

The `history` command display command history.

## Syntax

```bash
history [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
history

# Get help
history --help

# Check version
history --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `history` command is part of the `system-info` category in Linux systems. It provides essential functionality for display command history.

Refer to the manual page (`man history`) for detailed information about all available options and usage patterns.

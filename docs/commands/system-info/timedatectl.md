# timedatectl - Control system time and date



## Overview

The `timedatectl` command control system time and date.

## Syntax

```bash
timedatectl [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
timedatectl

# Get help
timedatectl --help

# Check version
timedatectl --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `timedatectl` command is part of the `system-info` category in Linux systems. It provides essential functionality for control system time and date.

Refer to the manual page (`man timedatectl`) for detailed information about all available options and usage patterns.

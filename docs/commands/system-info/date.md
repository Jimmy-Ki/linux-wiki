# date - Display system date and time



## Overview

The `date` command display system date and time.

## Syntax

```bash
date [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
date

# Get help
date --help

# Check version
date --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `date` command is part of the `system-info` category in Linux systems. It provides essential functionality for display system date and time.

Refer to the manual page (`man date`) for detailed information about all available options and usage patterns.

# time - Run command and display resource usage



## Overview

The `time` command run command and display resource usage.

## Syntax

```bash
time [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
time

# Get help
time --help

# Check version
time --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `time` command is part of the `system-info` category in Linux systems. It provides essential functionality for run command and display resource usage.

Refer to the manual page (`man time`) for detailed information about all available options and usage patterns.

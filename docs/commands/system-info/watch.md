# watch - Execute command periodically



## Overview

The `watch` command execute command periodically.

## Syntax

```bash
watch [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
watch

# Get help
watch --help

# Check version
watch --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `watch` command is part of the `system-info` category in Linux systems. It provides essential functionality for execute command periodically.

Refer to the manual page (`man watch`) for detailed information about all available options and usage patterns.

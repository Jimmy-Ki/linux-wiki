# mesg - Control write access to terminal



## Overview

The `mesg` command control write access to terminal.

## Syntax

```bash
mesg [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
mesg

# Get help
mesg --help

# Check version
mesg --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `mesg` command is part of the `system-info` category in Linux systems. It provides essential functionality for control write access to terminal.

Refer to the manual page (`man mesg`) for detailed information about all available options and usage patterns.

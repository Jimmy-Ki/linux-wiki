# System Information Commands

This section contains comprehensive documentation for Linux commands used to display system information, monitor processes, check system status, and gather various system metrics. These commands are essential for system administration, monitoring, and troubleshooting.

## Categories

- **System Information**: Display basic system details, kernel information, and hardware specs
- **Process Monitoring**: View running processes, system resources, and performance metrics
- **Network Statistics**: Monitor network connections, socket information, and network usage
- **Resource Usage**: Check memory, CPU, disk I/O, and system resource utilization

## Key Commands

- `uname` - Display system information
- `uptime` - Show system uptime and load average
- `ps` - Display process status
- `top` - Dynamic real-time process viewer
- `free` - Display memory usage
- `lsof` - List open files
- `netstat` - Network statistics
- `vmstat` - Virtual memory statistics

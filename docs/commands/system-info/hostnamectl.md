# hostnamectl - Control system hostname



## Overview

The `hostnamectl` command control system hostname.

## Syntax

```bash
hostnamectl [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
hostnamectl

# Get help
hostnamectl --help

# Check version
```

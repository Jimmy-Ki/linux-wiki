# lsmod - Show loaded kernel modules



## Overview

The `lsmod` command show loaded kernel modules.

## Syntax

```bash
lsmod [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
lsmod

# Get help
lsmod --help

# Check version
lsmod --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

The `lsmod` command is part of the `system-info` category in Linux systems. It provides essential functionality for show loaded kernel modules.

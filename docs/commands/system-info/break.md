# break - Exit from for, while, or until loops



## Overview

The `break` command exit from for, while, or until loops.

## Syntax

```bash
break [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
break

# Get help
break --help

# Check version
break --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `break` command is part of the `system-info` category in Linux systems. It provides essential functionality for exit from for, while, or until loops.

Refer to the manual page (`man break`) for detailed information about all available options and usage patterns.

# blkid - Locate block device attributes



## Overview

The `blkid` command locate block device attributes.

## Syntax

```bash
blkid [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
blkid

# Get help
blkid --help

# Check version
blkid --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `blkid` command is part of the `system-info` category in Linux systems. It provides essential functionality for locate block device attributes.

Refer to the manual page (`man blkid`) for detailed information about all available options and usage patterns.

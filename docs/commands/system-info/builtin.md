# builtin - Execute shell builtins



## Overview

The `builtin` command execute shell builtins.

## Syntax

```bash
builtin [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
builtin

# Get help
builtin --help

# Check version
builtin --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `builtin` command is part of the `system-info` category in Linux systems. It provides essential functionality for execute shell builtins.

Refer to the manual page (`man builtin`) for detailed information about all available options and usage patterns.

# alias - The `Alias` Command Is A Linux Utility For Alias Operations.



## Overview

The `alias` command create command aliases.

## Syntax

```bash
alias [-p] [name[=value] ...]
```

## Basic Usage Examples

```bash
# Basic usage
alias

# Get help
alias --help
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information

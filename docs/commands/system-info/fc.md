# fc - List or edit commands in history



## Overview

The `fc` command list or edit commands in history.

## Syntax

```bash
fc [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
fc

# Get help
fc --help

# Check version
fc --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

The `fc` command is part of the `system-info` category in Linux systems. It provides essential functionality for list or edit commands in history.

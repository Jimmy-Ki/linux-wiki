# iotop - Simple top-like I/O monitor



## Overview

The `iotop` command simple top-like i/o monitor.

## Syntax

```bash
iotop [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
iotop

# Get help
iotop --help

# Check version
iotop --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `iotop` command is part of the `system-info` category in Linux systems. It provides essential functionality for simple top-like i/o monitor.

Refer to the manual page (`man iotop`) for detailed information about all available options and usage patterns.

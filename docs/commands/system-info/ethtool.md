# ethtool - Display and modify network device settings



## Overview

The `ethtool` command display and modify network device settings.

## Syntax

```bash
ethtool [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
ethtool

# Get help
ethtool --help

# Check version
ethtool --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `ethtool` command is part of the `system-info` category in Linux systems. It provides essential functionality for display and modify network device settings.

Refer to the manual page (`man ethtool`) for detailed information about all available options and usage patterns.

# whatis - Display one-line manual page descriptions



## Overview

The `whatis` command display one-line manual page descriptions.

## Syntax

```bash
whatis [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
whatis

# Get help
whatis --help

# Check version
whatis --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `whatis` command is part of the `system-info` category in Linux systems. It provides essential functionality for display one-line manual page descriptions.

Refer to the manual page (`man whatis`) for detailed information about all available options and usage patterns.

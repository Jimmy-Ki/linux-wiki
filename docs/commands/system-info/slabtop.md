# slabtop - Display kernel slab cache information



## Overview

The `slabtop` command display kernel slab cache information.

## Syntax

```bash
slabtop [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
slabtop

# Get help
slabtop --help

# Check version
slabtop --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `slabtop` command is part of the `system-info` category in Linux systems. It provides essential functionality for display kernel slab cache information.

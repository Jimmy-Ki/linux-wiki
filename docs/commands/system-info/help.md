# help - Display help for shell builtins



## Overview

The `help` command display help for shell builtins.

## Syntax

```bash
help [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
help

# Get help
help --help

# Check version
help --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `help` command is part of the `system-info` category in Linux systems. It provides essential functionality for display help for shell builtins.

Refer to the manual page (`man help`) for detailed information about all available options and usage patterns.

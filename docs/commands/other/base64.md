# base64 - Encode and decode base64 data



## Overview

The `base64` command encode and decode base64 data.

## Syntax

```bash
base64 [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
base64

# Get help
base64 --help

# Check version
base64 --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

The `base64` command is part of the `other` category in Linux systems. It provides essential functionality for encode and decode base64 data.

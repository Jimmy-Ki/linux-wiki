# ispell - Interactive spelling checker



## Overview

The `ispell` command interactive spelling checker.

## Syntax

```bash
ispell [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
ispell

# Get help
ispell --help

# Check version
ispell --version
```

- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

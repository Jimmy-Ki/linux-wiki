# rsh - Remote shell



## Overview

The `rsh` command remote shell.

## Syntax

```bash
rsh [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage

# Get help
rsh --help

# Check version
rsh --version
```

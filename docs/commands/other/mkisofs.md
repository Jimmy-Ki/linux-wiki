# mkisofs - Create an ISO 9660/Joliet/HFS filesystem



## Overview

The `mkisofs` command create an iso 9660/joliet/hfs filesystem.

## Syntax

```bash
mkisofs [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash

mkisofs

# Get help
mkisofs --help

# Check version
```

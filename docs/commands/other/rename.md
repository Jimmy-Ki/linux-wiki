# rename - Rename files



## Overview

The `rename` command rename files.

## Syntax

```bash
rename [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
rename

# Get help
rename --help

# Check version
rename --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

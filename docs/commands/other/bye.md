# bye - Terminate ftp session and exit



## Overview

The `bye` command terminate ftp session and exit.

## Syntax

```bash
bye [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash

# Get help
bye --help

# Check version
bye --version
```

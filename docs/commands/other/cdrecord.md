# cdrecord - Record data to CD/DVD



## Overview

The `cdrecord` command record data to cd/dvd.

## Syntax

```bash
cdrecord [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
cdrecord

# Get help
cdrecord --help

# Check version
cdrecord --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

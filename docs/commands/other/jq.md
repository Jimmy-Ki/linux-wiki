# jq - Command-line JSON processor



## Overview

The `jq` command command-line json processor.

## Syntax

```bash
jq [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
jq

# Get help
jq --help

# Check version
jq --version
```

- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

# ar - The `Ar` Command Is A Linux Utility For Ar Operations.



## Overview

The `ar` command create, modify, and extract archives.

## Syntax

```bash
用法: ar [emulation options] [-]{dmpqrstx}[abcDfilMNoOPsSTuvV] [--plugin <name>] [member-name] [count] archive-file file...
      ar -M [<mri-脚本]
```

# clock - Get time and date



## Overview

The `clock` command get time and date.

## Syntax

```bash
clock [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage

# Get help
clock --help

# Check version
clock --version
```

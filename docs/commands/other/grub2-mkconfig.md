# grub2-mkconfig - Generate GRUB2 configuration file



## Overview

The `grub2-mkconfig` command generate grub2 configuration file.

## Syntax

```bash
grub2-mkconfig [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
grub2-mkconfig

# Get help
grub2-mkconfig --help

# Check version
grub2-mkconfig --version
```

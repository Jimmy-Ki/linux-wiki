# bzdiff - Compare bzip2 compressed files



## Overview

The `bzdiff` command compare bzip2 compressed files.

## Syntax

```bash
bzdiff [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
bzdiff

# Get help
bzdiff --help

# Check version
bzdiff --version
```

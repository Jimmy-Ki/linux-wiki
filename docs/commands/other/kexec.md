# kexec - Boot into new kernel



## Overview

The `kexec` command boot into new kernel.

## Syntax

```bash
kexec [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
kexec

# Get help
kexec --help

# Check version
kexec --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

The `kexec` command is part of the `other` category in Linux systems. It provides essential functionality for boot into new kernel.

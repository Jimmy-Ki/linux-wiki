# sosreport - Collect system configuration and diagnostic information



## Overview

The `sosreport` command collect system configuration and diagnostic information.

## Syntax

```bash
sosreport [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
sosreport

# Get help
sosreport --help

# Check version
sosreport --version
```

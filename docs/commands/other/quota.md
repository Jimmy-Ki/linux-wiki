# quota - Display disk usage and limits



## Overview

The `quota` command display disk usage and limits.

## Syntax

```bash
quota [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
quota

# Get help
quota --help

# Check version
quota --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

The `quota` command is part of the `other` category in Linux systems. It provides essential functionality for display disk usage and limits.

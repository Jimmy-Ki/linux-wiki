# env - Run command in modified environment



## Overview

The `env` command run command in modified environment.

## Syntax

```bash
env [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
env

# Get help
env --help

# Check version
env --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

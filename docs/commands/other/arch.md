# arch - The `Arch` Command Is A Linux Utility For Arch Operations.



## Overview

The `arch` command display system architecture.

## Syntax

```bash
arch [OPTION]...
```

## Basic Usage Examples

```bash
# Basic usage
arch

# Get help
arch --help
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information

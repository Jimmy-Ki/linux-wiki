# xrandr - Configure screen size and orientation



## Overview

The `xrandr` command configure screen size and orientation.

## Syntax

```bash
xrandr [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
xrandr

# Get help
xrandr --help

# Check version
xrandr --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `xrandr` command is part of the `other` category in Linux systems. It provides essential functionality for configure screen size and orientation.

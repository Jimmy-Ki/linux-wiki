# bzgrep - Search bzip2 compressed files



## Overview

The `bzgrep` command search bzip2 compressed files.

## Syntax

```bash
bzgrep [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
bzgrep

# Get help
bzgrep --help

# Check version
bzgrep --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

The `bzgrep` command is part of the `file-management` category in Linux systems. It provides essential functionality for search bzip2 compressed files.

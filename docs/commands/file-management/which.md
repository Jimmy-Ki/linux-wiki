# which - Locate command executable



## Overview

The `which` command locate command executable.

## Syntax

```bash
which [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
which

# Get help
which --help

# Check version
which --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `which` command is part of the `file-management` category in Linux systems. It provides essential functionality for locate command executable.

Refer to the manual page (`man which`) for detailed information about all available options and usage patterns.

# cmp - Compare two files byte by byte



## Overview

The `cmp` command compare two files byte by byte.

## Syntax

```bash
cmp [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
cmp

# Get help
cmp --help

# Check version
cmp --version
```

- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

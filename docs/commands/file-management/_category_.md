# File Management Commands

This section contains commands for file operations, directory management, file searching, and file manipulation.

## Commands Included:
- **diff** - Compare files line by line
- **diffstat** - Show statistics from diff output
- **file** - Determine file type
- **find** - Search for files
- **git** - Distributed version control system
- **gitview** - Git repository viewer
- **ln** - Create links between files
- **locate** - Find files by name
- **lsattr** - List file attributes
- **mattrib** - Change MSDOS file attributes
- **mc** - Midnight Commander file manager
- **mcopy** - Copy MSDOS files
- **mdel** - Delete MSDOS files
- **mdir** - Display MSDOS directory
- **mktemp** - Create temporary files
- **mmove** - Move MSDOS files
- **mread** - Read MSDOS files
- **mren** - Rename MSDOS files
- **mshowfat** - Show FAT clusters
- **mtools** - MSDOS utilities
- **mtoolstest** - Test mtools
- **mv** - Move or rename files
- **od** - Octal dump
- **paste** - Merge lines of files
- **patch** - Apply patch files
- **rcp** - Remote copy
- **rhmask** - Generate RHNet masks
- **rm** - Remove files
- **slocate** - Secure file locate
- **split** - Split files into pieces
- **tee** - Read from standard write to standard output and files
- **tmpwatch** - Remove temporary files
- **touch** - Change file timestamps
- **umask** - Set file creation mask
- **whereis** - Locate programs
- **which** - Locate a program
- **cat** - Concatenate files
- **chattr** - Change file attributes
- **chgrp** - Change group ownership
- **chmod** - Change file permissions
- **chown** - Change file ownership
- **cksum** - Calculate CRC checksum
- **cmp** - Compare files
- **cp** - Copy files
- **cut** - Remove sections from each line
- **indent** - Change C program indentation

# bzcat - Decompress bzip2 files to stdout



## Overview

The `bzcat` command decompress bzip2 files to stdout.

## Syntax

```bash
bzcat [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
bzcat

# Get help
bzcat --help

# Check version
bzcat --version
```

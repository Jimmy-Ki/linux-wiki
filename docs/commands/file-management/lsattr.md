# lsattr - List file attributes on Linux



## Overview

The `lsattr` command list file attributes on linux.

## Syntax

```bash
lsattr [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
lsattr

# Get help
lsattr --help

# Check version
lsattr --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

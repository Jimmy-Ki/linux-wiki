# unexpand - Convert spaces to tabs



## Overview

The `unexpand` command convert spaces to tabs.

## Syntax

```bash
unexpand [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
unexpand

# Get help
unexpand --help

# Check version
unexpand --version
```

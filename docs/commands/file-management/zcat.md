# zcat - Decompress files to stdout



## Overview

The `zcat` command decompress files to stdout.

## Syntax

```bash
zcat [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
zcat

# Get help
zcat --help

# Check version
zcat --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

# logout - Exit login shell



## Overview

The `logout` command exit login shell.

## Syntax

```bash
logout [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
logout

# Get help
logout --help

# Check version
logout --version
```

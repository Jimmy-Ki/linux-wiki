# users - Display current user names



## Overview

The `users` command display current user names.

## Syntax

```bash
users [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
users

# Get help
users --help

# Check version
users --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

The `users` command is part of the `user-management` category in Linux systems. It provides essential functionality for display current user names.

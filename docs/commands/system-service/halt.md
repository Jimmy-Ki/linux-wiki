# halt - Stop the system



## Overview

The `halt` command stop the system.

## Syntax

```bash
halt [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
halt

# Get help
halt --help

# Check version
halt --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `halt` command is part of the `system-service` category in Linux systems. It provides essential functionality for stop the system.

Refer to the manual page (`man halt`) for detailed information about all available options and usage patterns.

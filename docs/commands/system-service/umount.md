# umount - Unmount filesystems



## Overview

The `umount` command unmount filesystems.

## Syntax

```bash
umount [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
umount

# Get help
umount --help

# Check version
umount --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `umount` command is part of the `system-service` category in Linux systems. It provides essential functionality for unmount filesystems.

Refer to the manual page (`man umount`) for detailed information about all available options and usage patterns.

# pvremove - Remove physical volume



## Overview

The `pvremove` command remove physical volume.

## Syntax

```bash
pvremove [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
pvremove

# Get help
pvremove --help

# Check version
pvremove --version
```

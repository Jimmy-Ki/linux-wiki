# pvscan - Scan all disks for physical volumes



## Overview

The `pvscan` command scan all disks for physical volumes.

## Syntax

```bash
pvscan [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
pvscan

# Get help
pvscan --help

# Check version
pvscan --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

The `pvscan` command is part of the `system-service` category in Linux systems. It provides essential functionality for scan all disks for physical volumes.

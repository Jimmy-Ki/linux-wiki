# insmod - Insert module into Linux kernel



## Overview

The `insmod` command insert module into linux kernel.

## Syntax

```bash
insmod [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
insmod

# Get help
insmod --help

# Check version
insmod --version
```

- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

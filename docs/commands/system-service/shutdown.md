# shutdown - Bring the system down



## Overview

The `shutdown` command bring the system down.

## Syntax

```bash
shutdown [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
shutdown

# Get help
shutdown --help

# Check version
shutdown --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `shutdown` command is part of the `system-service` category in Linux systems. It provides essential functionality for bring the system down.

Refer to the manual page (`man shutdown`) for detailed information about all available options and usage patterns.

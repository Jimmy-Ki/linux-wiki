# pvdisplay - Display attributes of physical volume



## Overview

The `pvdisplay` command display attributes of physical volume.

## Syntax

```bash
pvdisplay [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
pvdisplay

# Get help
pvdisplay --help

# Check version
pvdisplay --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `pvdisplay` command is part of the `system-service` category in Linux systems. It provides essential functionality for display attributes of physical volume.

Refer to the manual page (`man pvdisplay`) for detailed information about all available options and usage patterns.

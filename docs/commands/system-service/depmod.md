# depmod - Generate modules.dep and map files



## Overview

The `depmod` command generate modules.dep and map files.

## Syntax

```bash
depmod [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
depmod

# Get help
depmod --help

# Check version
depmod --version
```

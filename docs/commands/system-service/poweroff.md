# poweroff - Stop the system



## Overview

The `poweroff` command stop the system.

## Syntax

```bash
poweroff [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
poweroff

# Get help
poweroff --help

# Check version
poweroff --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `poweroff` command is part of the `system-service` category in Linux systems. It provides essential functionality for stop the system.

Refer to the manual page (`man poweroff`) for detailed information about all available options and usage patterns.

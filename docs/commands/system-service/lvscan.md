# lvscan - Scan for logical volumes



## Overview

The `lvscan` command scan for logical volumes.

## Syntax

```bash
lvscan [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
lvscan

# Get help
lvscan --help

# Check version
lvscan --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `lvscan` command is part of the `system-service` category in Linux systems. It provides essential functionality for scan for logical volumes.

Refer to the manual page (`man lvscan`) for detailed information about all available options and usage patterns.

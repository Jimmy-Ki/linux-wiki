# vgscan - Scan for volume groups



## Overview

The `vgscan` command scan for volume groups.

## Syntax

```bash
vgscan [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
vgscan

# Get help
vgscan --help

# Check version
vgscan --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `vgscan` command is part of the `system-service` category in Linux systems. It provides essential functionality for scan for volume groups.

Refer to the manual page (`man vgscan`) for detailed information about all available options and usage patterns.

# fdisk - Disk partitioning utility



## Overview

The `fdisk` command disk partitioning utility.

## Syntax

```bash
fdisk [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
fdisk

# Get help
fdisk --help

# Check version
fdisk --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `fdisk` command is part of the `system-service` category in Linux systems. It provides essential functionality for disk partitioning utility.

Refer to the manual page (`man fdisk`) for detailed information about all available options and usage patterns.

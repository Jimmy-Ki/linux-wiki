# iperf - Network bandwidth measurement tool



## Overview

The `iperf` command network bandwidth measurement tool.

## Syntax

```bash
iperf [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
iperf

# Get help
iperf --help

# Check version
iperf --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `iperf` command is part of the `networking` category in Linux systems. It provides essential functionality for network bandwidth measurement tool.

Refer to the manual page (`man iperf`) for detailed information about all available options and usage patterns.

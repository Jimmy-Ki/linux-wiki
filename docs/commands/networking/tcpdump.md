# tcpdump - Dump traffic on network



## Overview

The `tcpdump` command dump traffic on network.

## Syntax

```bash
tcpdump [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
tcpdump

# Get help
tcpdump --help

# Check version
tcpdump --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `tcpdump` command is part of the `networking` category in Linux systems. It provides essential functionality for dump traffic on network.

Refer to the manual page (`man tcpdump`) for detailed information about all available options and usage patterns.

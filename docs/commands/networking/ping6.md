# ping6 - Test IPv6 network connectivity



## Overview

The `ping6` command test ipv6 network connectivity.

## Syntax

```bash
ping6 [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
ping6

# Get help
ping6 --help

# Check version
ping6 --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `ping6` command is part of the `networking` category in Linux systems. It provides essential functionality for test ipv6 network connectivity.

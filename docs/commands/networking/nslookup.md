# nslookup - Query Internet name servers interactively



## Overview

The `nslookup` command query internet name servers interactively.

## Syntax

```bash
nslookup [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
nslookup

# Get help
nslookup --help

# Check version
nslookup --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `nslookup` command is part of the `networking` category in Linux systems. It provides essential functionality for query internet name servers interactively.

Refer to the manual page (`man nslookup`) for detailed information about all available options and usage patterns.

# sshpass - Non-interactive ssh password provider



## Overview

The `sshpass` command non-interactive ssh password provider.

## Syntax

```bash
sshpass [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
sshpass

# Get help
sshpass --help

# Check version
sshpass --version
```

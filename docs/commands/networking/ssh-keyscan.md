# ssh-keyscan - Gather SSH public keys



## Overview

The `ssh-keyscan` command gather ssh public keys.

## Syntax

```bash
ssh-keyscan [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
ssh-keyscan

# Get help
ssh-keyscan --help

# Check version
ssh-keyscan --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

## Description

The `ssh-keyscan` command is part of the `networking` category in Linux systems. It provides essential functionality for gather ssh public keys.

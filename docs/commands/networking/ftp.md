# ftp - File Transfer Protocol client



## Overview

The `ftp` command file transfer protocol client.

## Syntax

```bash
ftp [OPTIONS] [ARGUMENTS]
```

## Basic Usage Examples

```bash
# Basic usage
ftp

# Get help
ftp --help

# Check version
ftp --version
```

## Common Options

- `-h, --help` - Display help information
- `-v, --version` - Show version information
- `-V, --verbose` - Enable verbose output (if supported)

The `ftp` command is part of the `networking` category in Linux systems. It provides essential functionality for file transfer protocol client.

Refer to the manual page (`man ftp`) for detailed information about all available options and usage patterns.
